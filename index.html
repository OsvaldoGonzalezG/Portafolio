Caso de estudio · Myth Store

Proyecto elegido para representar mi crecimiento técnico y mi capacidad de construir una solución funcional y presentable.

Descripción

Myth Store es una aplicación web orientada a la exhibición de productos TCG. El objetivo fue construir una experiencia clara para visualizar un catálogo, presentar productos y trabajar una identidad visual coherente.

Desafío principal

Organizar la información, estructurar el catálogo y lograr una interfaz más atractiva sin perder claridad.

Solución aplicada

Se desarrolló una base web con Django, templates HTML, estilos CSS y apoyo de Bootstrap para construir una navegación funcional y ordenada.

Aprendizajes

Modelado de datos, relación entre backend y frontend, estructura de proyectos y mejora visual orientada a portafolio.

Herramientas y valor técnico

Stack

Python, Django, HTML, CSS, Bootstrap y SQL.

Habilidades aplicadas

Desarrollo backend, diseño de vistas, templates, organización visual, estructura de catálogo y lógica de aplicación web.

Por qué lo elegí

Porque integra desarrollo, estructura, diseño y una identidad propia. Resume bien mi evolución técnica y mi capacidad de transformar una idea en un producto visible.
